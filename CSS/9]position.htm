position: static/relative/ absolute/fixed/sticky

static- The element is positioned according to the normal flow of the document. The top, right, bottom, left, and z-index properties have no effect. This is the default value.

relative- It is position with respect to the position given by div tag. It get affected by z-idex

absolute- The absolutely positioned element is positioned relative to its nearest positioned ancestor i.e., the nearest ancestor that is not static. hat element get removed from document flow. and element can move out of the screen while scrolling

fixed-It the position wrt the browser window. hat element get removed from document flow. and element can't move out of the screen while scrolling

By putting relative position to outside div tag and absolute position to inner div tag we can fix the position of inner div tag element wrt outer div tag.
